# STM32-1602a

Driver for 1602a LCD module connected to STM32f103rb

 RS  - GPIOB_PIN_5 <br>
 RW  - Low
 E   - GPIOA_PIN_8 <br>
 DB4 - GPIOB_PIN_4<br>
 DB5 - GPIOB_PIN_0<br>
 DB6 - GPIOC_PIN_1<br>
 DB7 - GPIOC_PIN_0<br>
